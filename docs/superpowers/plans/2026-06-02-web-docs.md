# Web Docs Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Web アプリ前提の技術選定を明文化し、`README.md` と `CODEX.md` を作成する。

**Architecture:** 人間開発者向けの `README.md` と、AI エージェント向けの `CODEX.md` に責務を分ける。両方とも `docs/superpowers/specs/2026-06-02-web-docs-design.md` の合意内容に沿って、個人利用向けハイブリッド Web アプリとして記述する。

**Tech Stack:** Next.js, React, TypeScript, Tailwind CSS, Python worker, ffmpeg, PaddleOCR or EasyOCR, Pillow, local file storage.
